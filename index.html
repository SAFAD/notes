4.2 create one fat jar for a jersey 2 application (solution found at Jersey exception only thrown when depencencies assembled into a single jar)

<plugin>
    <groupId>org.apache.maven.plugins</groupId>
    <artifactId>maven-shade-plugin</artifactId>
    <version>2.3</version>
    <executions>
        <execution>
            <phase>package</phase>
                <goals>
                    <goal>shade</goal>
                </goals>
            <configuration>
                <transformers>
                    <transformer implementation="org.apache.maven.plugins.shade.resource.ServicesResourceTransformer" />
                    <transformer implementation="org.apache.maven.plugins.shade.resource.ManifestResourceTransformer">
                        <mainClass>${mainClass}</mainClass>
                    </transformer>
                </transformers>
                <filters>
                    <filter>
                        <artifact>*:*</artifact>
                        <excludes>
                            <exclude>META-INF/*.SF</exclude>
                            <exclude>META-INF/*.DSA</exclude>
                            <exclude>META-INF/*.RSA</exclude>
                        </excludes>
                    </filter>
                </filters>
                <finalName>${fat-jar-name}</finalName>\
            </configuration>
        </execution>
    </executions>
</plugin>

4.3 renanme a jar-with-dependencies file created by maven-assembly-plugin

add this two elements to the maven-assembly-plugin configuration element: 

<configuration>
    ...
    <finalName>fat-jar-name</finalName>
    <appendAssemblyId>false</appendAssemblyId>
</configuration>

8.6 rsfs fonts for math

  • install the package via: tlmgr install rsfs.
  • import the package in the preamble: 
\usepackage{mathrsfs}
\DeclareMathAlphabet{\mathpzc}{OT1}{pzc}{m}{it}
  • use it! 
$mathpzc{uppercase or lowercase characters}$

8.9 inline enumerate items with a modified label

  • a link to the source.
  • use enumitem package with inline option to enable inlined enumeration command (enumerate*). 
\usepackage[inline]{enumitem}
  • define `inlinelist` command: 
\newlist{inlinelist}{enumerate*}{1}
\setlist*[inlinelist,1]{%
  label=(\roman*),
}
